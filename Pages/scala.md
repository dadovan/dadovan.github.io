## Collections

### List

Create a new immuatable list:

```scala
val weekend = List("Sat", "Sun")
```

### Map

Create a new immuatable map (notice the two ways to specify an entry):

```scala
val weekend = Map("Mon" -> "No", ("Fri", "Yes"))
```
